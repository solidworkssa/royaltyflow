# RoyaltyFlow

Automated creator royalty distribution system.

## Features
- Multi-chain support (Base & Stacks)
- Secure Contract Logic
- React/Next.js Frontend

## Contracts
- Base: `contracts/base/src/RoyaltyFlow.sol`
- Stacks: `contracts/stacks/contracts/royaltyflow.clar`

## Setup
```bash
pnpm install
pnpm dev
```
